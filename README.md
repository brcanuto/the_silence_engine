## Overview

**The Silence Engine** is a web-based puzzle application built as a proof-of-concept for combining:

- **Python (FastAPI) backend**  
- **React (Vite) frontend**  
- **Modular, reusable puzzle logic**  
- **Clean API-driven architecture**

- Python 3.10+
- FastAPI
- Uvicorn
- Pydantic

### **Frontend**
- React (Vite)
- JavaScript / JSX
- Fetch API
